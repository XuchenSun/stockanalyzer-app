常见问题

如何使用AI分析功能？
在股票详情页面，点击"AI分析"按钮即可获取AI对该股票的深度分析报告。分析内容包括技术面分析、基本面概览和投资建议。需要有效的网络连接。
数据多久更新一次？
股票行情数据通过 Finnhub API 实时获取。在交易时段内，数据会随您的查看请求实时刷新。历史数据可查看多个时段周期。
如何管理订阅？
打开iOS设置 → 点击您的Apple ID → 订阅 → 找到"股智 StockPulse AI"进行管理。您可以在此更改或取消订阅计划。
支持哪些股票市场？
通过 Finnhub API 支持全球主要市场股票，包括美股（NYSE、NASDAQ）、港股等。搜索时使用对应的股票代码，例如：AAPL（苹果）、0700.HK（腾讯）。
应用无法加载数据怎么办？
请尝试以下步骤：1) 检查网络连接是否正常；2) 确认非交易时段可能无实时数据；3) 尝试关闭并重新打开应用；4) 检查是否有可用的应用更新。如问题持续，请通过下方邮箱联系我们。
如何设置价格提醒？
打开任意股票详情页，点击右上角铃铛图标，输入目标价格并选择"高于"或"低于"触发方向，点击"添加提醒"即可。价格提醒为 Pro 专属功能，需要允许通知权限。

Frequently Asked Questions

How do I use the AI analysis feature?
On the stock detail page, tap the "AI Analysis" button to get an in-depth AI analysis report. The analysis includes technical analysis, fundamental overview, and investment insights. An active internet connection is required.
How often is data updated?
Stock market data is fetched in real-time via the Finnhub API. During trading hours, data refreshes with each viewing request. Historical data is available across multiple chart periods.
How do I manage my subscription?
Open iOS Settings → Tap your Apple ID → Subscriptions → Find "StockPulse AI" to manage. You can change or cancel your subscription plan here.
Which stock markets are supported?
We support major global markets via the Finnhub API, including US stocks (NYSE, NASDAQ), Hong Kong stocks, and more. Search using the ticker symbol, e.g., AAPL (Apple), 0700.HK (Tencent).
The app can't load data — what should I do?
Try these steps: 1) Check your internet connection; 2) Note that real-time data may be unavailable outside trading hours; 3) Close and reopen the app; 4) Check for available app updates. If the issue persists, contact us via email below.
How do I set a price alert?
Open any stock detail page and tap the bell icon in the top-right corner. Enter your target price, choose "Above" or "Below" as the trigger direction, then tap "Add Alert". Price alerts are a Pro feature and require notification permissions to be enabled.

Preguntas Frecuentes

¿Cómo uso la función de análisis IA?
En la página de detalles de la acción, toca el botón "Análisis IA" para obtener un informe de análisis en profundidad. El análisis incluye análisis técnico, resumen fundamental y perspectivas de inversión. Se requiere conexión a internet activa.
¿Con qué frecuencia se actualizan los datos?
Los datos bursátiles se obtienen en tiempo real mediante la API de Finnhub. Durante el horario de negociación, los datos se actualizan con cada solicitud de visualización. Los datos históricos están disponibles en múltiples períodos gráficos.
¿Cómo gestiono mi suscripción?
Abre Ajustes de iOS → Toca tu Apple ID → Suscripciones → Busca "StockPulse AI" para gestionar. Aquí puedes cambiar o cancelar tu plan de suscripción.
¿Qué mercados bursátiles son compatibles?
Soportamos los principales mercados globales mediante la API de Finnhub, incluyendo acciones de EE.UU. (NYSE, NASDAQ), acciones de Hong Kong y más. Busca usando el símbolo de cotización, por ejemplo, AAPL (Apple), 0700.HK (Tencent).
La app no carga datos — ¿qué debo hacer?
Prueba estos pasos: 1) Verifica tu conexión a internet; 2) Ten en cuenta que los datos en tiempo real pueden no estar disponibles fuera del horario de negociación; 3) Cierra y vuelve a abrir la app; 4) Busca actualizaciones disponibles. Si el problema persiste, contáctanos por correo electrónico.
¿Cómo configuro una alerta de precio?
Abre cualquier página de detalles de una acción y toca el icono de campana en la esquina superior derecha. Ingresa tu precio objetivo, elige "Por encima" o "Por debajo" como dirección de activación y toca "Agregar alerta". Las alertas de precio son una función Pro y requieren permisos de notificación habilitados.

Foire Aux Questions

Comment utiliser la fonction d'analyse IA ?
Sur la page de détails d'une action, appuyez sur le bouton "Analyse IA" pour obtenir un rapport d'analyse approfondi. L'analyse comprend l'analyse technique, un aperçu fondamental et des perspectives d'investissement. Une connexion internet active est requise.
À quelle fréquence les données sont-elles mises à jour ?
Les données boursières sont récupérées en temps réel via l'API Finnhub. Pendant les heures de négociation, les données se rafraîchissent à chaque demande d'affichage. Les données historiques sont disponibles sur plusieurs périodes graphiques.
Comment gérer mon abonnement ?
Ouvrez les Réglages iOS → Appuyez sur votre Apple ID → Abonnements → Trouvez "StockPulse AI" pour gérer. Vous pouvez modifier ou annuler votre abonnement ici.
Quels marchés boursiers sont pris en charge ?
Nous supportons les principaux marchés mondiaux via l'API Finnhub, notamment les actions américaines (NYSE, NASDAQ), les actions de Hong Kong et plus encore. Recherchez en utilisant le symbole boursier, par ex. AAPL (Apple), 0700.HK (Tencent).
L'application ne charge pas les données — que faire ?
Essayez ces étapes : 1) Vérifiez votre connexion internet ; 2) Notez que les données en temps réel peuvent être indisponibles en dehors des heures de négociation ; 3) Fermez et rouvrez l'application ; 4) Recherchez les mises à jour disponibles. Si le problème persiste, contactez-nous par e-mail.
Comment configurer une alerte de prix ?
Ouvrez n'importe quelle page de détails d'action et appuyez sur l'icône de cloche en haut à droite. Saisissez votre prix cible, choisissez "Au-dessus" ou "En dessous" comme direction de déclenchement, puis appuyez sur "Ajouter une alerte". Les alertes de prix sont une fonction Pro et nécessitent l'activation des permissions de notifications.
